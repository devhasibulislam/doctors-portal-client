## For Server Side:
* Express
* Cors
* Body-Parser
* MongoDB
* JWT
* DotEnv
* NodeMon

**To install them, copy following:**
```bash
npm i express cors body-parser mongodb jsonwebtoken jwt dotenv nodemon
```
